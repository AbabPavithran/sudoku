# Sudoku

A classic 9x9 Sudoku web application.

## Features
- Multiple difficulty levels (Easy, Medium, Hard, Expert)
- Move history and undo support
- Candidate pencil notes mode
- Audio feedback and completion animations
- Game stats (timer & mistake counter)
