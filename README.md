# 🧩 Sudoku - Modern Interactive Web Game

An elegant, feature-rich, interactive Sudoku puzzle game built with HTML5, CSS3, and JavaScript.

## ✨ Features
- 🌲 **Dark Emerald Theme**: Sleek dark mode palette with geometric typography (`Outfit` font).
- ⚡ **MRV Backtracking Solver & Generator**: Instant puzzle creation across 4 difficulty levels (*Easy*, *Medium*, *Hard*, *Expert*).
- 🎵 **Web Audio API Sound Effects**: Procedural sound feedback for number placement, error alerts, and line completion shine effects.
- ✨ **Line & 3x3 Block Completion Shine**: Radiant glowing animation whenever a row, column, or 3x3 block is filled.
- ✏️ **Pencil Notes Mode**: Candidate number tracking within empty cells.
- ⏱️ **Timer & Game Over System**: Timer starts strictly on user play; includes max mistake tracking (3/3) and win celebratory modals.
- ⌨️ **Full Keyboard & Touch Support**: Intuitive navigation with Arrow keys, 1-9 keys, Backspace, N (Pencil), Ctrl+Z (Undo), P (Pause), and H (Hint).
